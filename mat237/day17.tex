\section{Day 17: Directional Derivatives (Oct 09, 2025)}

\begin{definition}[Directional Derivative]
\noindent This is a vector quantity. The directional derivative of $f$ at $a$ in the direction $v$ is given by
\[
    D_v f(a) = \lim_{ h \to 0 } \frac{f(a + hv) - f(a)}{h}
\]
\end{definition}

\noindent It can be shown that $D_vf(a)$ exists if and only if $D_vf_i(a)$ exists for $i \in \{ 1, \cdots, m \}$.
