\section{Day 34: Higher order partial derivatives (Nov 27, 2025)}

The order of some partial derivative $\partial_{i_1} \cdots \partial_{i_k} f$, where each $i_\cdot \in \{ 1, \cdots, n \}$, is given by $k$.
\begin{theorem}[Clairaut]
    Let $U \subseteq \mathbb{R}^n$ be open. If $f : U \to \mathbb{R}^m$ is $C^k$, then for any reordering $\{ j\}_1^k$ of $\{ i\}_1^k$,
    \[
    \partial_{i_1} \cdots \partial_{i_k} f = \partial_{j_1} \cdots \partial_{j_k} f
    \]
\end{theorem}
